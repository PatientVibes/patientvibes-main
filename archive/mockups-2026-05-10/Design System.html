00 · Foundations
A quietly opinionated
system for shipping agents.

Editorial typography on a calm sage canvas. Hairlines, generous space, deliberate accents. Built for developers — every component reads like documentation, every surface earns its weight.

Principles · 01
Quiet by default

No gradients without reason. No shadow without cause. Hairlines and ink before fills.

Principles · 02
Editorial pacing

Serif display sets the tone; sans carries the work; mono carries the truth. Three voices, never four.

Principles · 03
Truth in mono

Anything the machine knows — IDs, costs, latencies, code — wears JetBrains Mono. The eye learns to trust it.

Defining an agent.

An agent is a function with tools, a budget, and a model. The runtime handles tracing, retries, and deploys.

A minimal agent

Start with defineAgent. The model field is required; everything else has a sensible default.

07 · Voice & usage

Read like a developer wrote it.

Direct. Curious. A little dry. Never breathless. Avoid the verbs every AI startup uses.

Voice — do
✓

Show the artifact. "1.84s · $0.0042" beats "blazingly fast and affordable" every time.

✓

Name the tradeoff. If something is opinionated, say what you traded away to get it.

✓

Lowercase the technical. triage.v3, claude-haiku-4-5 — these are filenames, not titles.

✓

One italic per page. Reserve it for the word that carries weight.

Voice — don't
✕

Avoid superlatives. "Powerful," "intelligent," "revolutionary" — strip them.

✕

No emoji. Use a sage dot or a mono symbol instead.

✕

No gradients for decoration. Gradients carry data (throughput, cost trend) or they don't appear.

✕

No drop shadows on text. Ever.

Layout — do
✓

Pair a serif display with mono eyebrows. The eye learns the cadence.

✓

Use the ink rule for the top of the page; hairlines everywhere else.

✓

Leave the right column negative on hero sections — let the page breathe.

Layout — don't
✕

Don't stack three different border-radii in the same view.

✕

Don't use the terminal as a generic dark card. It earns its weight by being rare.

✕

Don't decorate empty space. White space is content.
